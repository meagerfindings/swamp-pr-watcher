# swamp-pr-watcher

[swamp](https://swamp.club) extension `@mgreten/pr-watcher` — an autonomous
pull-request feedback investigation engine that investigates PR feedback with a
coding agent, notifies via ntfy with an Approve button, and can apply approved
fixes inside an isolated worktree.

📖 **Extension documentation:** [extensions/models/README.md](extensions/models/README.md)

📦 **Install:** `swamp extension pull @mgreten/pr-watcher`

## Repository layout

This repository is a swamp workspace (its own `.swamp.yaml` repo). The
publishable extension lives under [extensions/models/](extensions/models/).
